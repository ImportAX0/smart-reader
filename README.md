# 智能阅读器 Smart Reader
个性化文本视觉增强工具（ADHD-friendly reading experience）, 通过视觉增强技术提升阅读效率的工具

## 🔗 Online Demo
**Reader (GitHub Pages):**  
https://importax0.github.io/smart-reader/

> Tip: 建议 Ctrl+Click / Cmd+Click 在新标签打开（不会离开 GitHub 页面）
## 功能特点

-  支持TXT文件拖拽上传
-  智能词性识别与视觉增强
-  纯净阅读模式切换
-  字体、字号、背景自定义
-  中英文混合支持
